What's New in the ABL Extension 1.32.0

Important update: new commercial features and licensing

The OpenEdge ABL extension has grown significantly over the past years, and so have the development and maintenance costs associated with it. To ensure the continued delivery of new features, bug fixes, and long-term support, a commercial licensing model is being introduced for selected advanced capabilities. Core code editing features remain, and will always remain, available at no cost.

Recent Highlights

Here is a summary of the major features delivered recently:

  • Code formatter: full document and range formatting, based on PDSOE
  • SpeedScript / Webspeed support: syntax highlighting and language features for Webspeed files
  • PASOE Debugger improvements: referenced projects, SSL communication, resume per thread
  • Classic Debugger improvements: improved source code search, launch mode fixes
  • Maven-based dependency management: including transitive dependencies
  • ABL Outline View & Class Browser: navigate your code structure at a glance
  • Organize USINGs: clean up USING statements automatically (Ctrl+Shift+U)
  • Expand Keywords: expand abbreviated keywords to their full form
  • Right-click compile: compile files and directories directly from the Explorer view
  • Online documentation: links to Progress documentation on statement hover
  • Pre- and post-compilation hooks: run scripts before and after each build
  • OpenEdge 13.0 support: documentation and runtime compatibility

What's Free, What's Not

Starting from version 1.32, some advanced features will require a paid subscription. All core code editing features remain free forever.

Always free

  • Syntax completion & hover
  • Go-to-definition & references
  • Code formatting
  • Compile, XREF, listing generation
  • Pre- and post-compilation hooks
  • openedge-project.json support
  • Data Dictionary integration
  • AppBuilder / Procedure Editor integration
  • Run TTY, Batch, and GUI procedures
  • Organize USINGs, Expand Keywords

Requires a license (from v1.32)

  • PASOE Debugger
  • Classic Debugger (_progres or prowin)
  • SpeedScript / Webspeed support
  • Maven-based dependency management
  • Email-based support (support@riverside-software.fr)
